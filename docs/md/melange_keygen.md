## melange keygen

Generate a key for package signing

### Synopsis

Generate a key for package signing.

```sh
$ melange keygen [flags]
```

### Examples

```sh
$ melange keygen [key.rsa]
```

### Options

```
  -h, --help           help for keygen
      --key-size int   the size of the prime to calculate (in bits) (default 4096)
```
